Question & Answer Part

1) What is DOM and what is the purpose of DOM?

Answer:

DOM is programming interface of XML or HTML document.We can dynamically access the document structure,style,content using DOM.Also we can access our page easily and modify our page when needed.We can create element,remove element,modify element,attributes modify,styling add or remove, create event using DOM.


2) How will you select HTML elements using DOM? Name the DOM methods.

Answer:

We can select HTML elements using DOM.There are 5 ways to select element in DOM selector.
1. getElementsByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

3) What is event bubble?

Answer:

Event bubble is a way that events are handled in the browser.It happens when an element receives an element that events bubbles up to the parent.If an event handler is set for the parent,this event handler is triggered.

4) What is a callback function and why will you use it?

Answer:

A callback is a function passed as an argument to another function.Callback functions are needed because many JavaScript actions are same.
